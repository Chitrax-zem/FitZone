A full-stack web application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) that digitizes gym operations and enhances member experience through automated membership management, trainer booking, and class scheduling.

Key Features
Membership Plans: Browse and purchase tiered membership packages (Basic, Premium, Elite)
Trainer Booking: View trainer profiles with specializations, ratings, and book personal training sessions
Class Scheduling: Interactive calendar for booking group fitness classes with real-time availability
User Dashboard: Track workouts, manage subscriptions, view bookings, and monitor fitness progress
Secure Authentication: JWT-based login system with role-based access control
Payment Integration: Multiple payment options including cards, UPI, and online wallets
Offline Support: Local storage fallback ensures functionality during connectivity issues
Responsive Design: Mobile-first approach works seamlessly across all devices
Tech Stack
Frontend: React.js, Tailwind CSS, Axios, Context API
Backend: Node.js, Express.js, JWT, Bcrypt
Database: MongoDB with Mongoose ODM
Features: RESTful APIs, Real-time booking, Payment gateway integration

Core Modules
User Management - Registration, authentication, profile management
Membership System - Plan selection, subscription purchase, renewal management
Trainer System - Profile browsing, availability checking, session booking
Class System - Schedule viewing, enrollment, capacity management
Booking System - Real-time reservations, cancellations, waitlists
Dashboard - Activity tracking, statistics, upcoming events
Benefits
Streamlines gym operations and reduces manual work
Enhances member engagement and satisfaction
Provides data-driven insights for business decisions
Scalable solution for single or multi-location gyms
24/7 accessibility for bookings and management
This modern platform transforms traditional gym management into an efficient, user-friendly digital experience for both gym operators and members.
